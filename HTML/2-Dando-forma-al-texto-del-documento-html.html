Si queremos separar los párrafos, o cualquier otra cosa, pero sin dejar una línea en blanco como a continuacion:
Usamos una etiqueta pensada para ello: br (break, o romper). No tiene etiqueta de cierre. Al escribir el texto, si ponemos más de un espacio en blanco entre dos palabras observamos que el navegador sólo reconoce uno de ellos. Si queremos forzarle a que lo haga, debemos poner el código  (non-breaking space)

B Para poner algo en negrita (bold).
I Para poner algo en cursiva (italic)

Otra etiqueta interesante es PRE. El texto que se encuentre entre ella estará preformateado, es decir que aparecerá como haya sido escrito y con una fuente de espaciado fijo (tipo Courier). Se respetarán los espacios en blanco y retornos del carro, tal como estaban en nuestro documento HTML. Es muy apropiada para confeccionar tablas poco elaboradas y otros documentos similares. 
"Texto de ejemplo de la etiqueta PRE",a comtinuacion insertare un salto de linea:

Con la etiqueta TT conseguimos que el texto tenga un tamaño menor y la apariencia de los caracteres de una máquina de escribir (typewriter). La diferencia con la anterior es que no preformatea el texto, sino que únicamente cambia su apariencia.
"Texto de ejemplo con la etiqueta TT",a comtinuacion insertare un salto de linea: He dado un salto de linea.

La etiqueta BLOCKQUOTE se utiliza para destacar una cita textual dentro del texto general. Este párrafo está escrito entre ambas etiquetas de inicio y cierre. Deja márgenes a ambos lados, por eso se usa para poner sangrías.
